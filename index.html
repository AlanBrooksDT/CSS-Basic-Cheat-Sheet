CSS & HTML Cheat Sheet

CSS Selectors

Selector Purpose
* Universal selector for all HTML elements
p Selector for all paragraph elements
.class Selector for element with class of 'class'
#id Selector for element with id of 'id'
p, h1, h2 How to select multiple elements to apply the same style
div p Descendant Combinator - Selector for any p elements within a div element
div > p Child Combinator - Selector for any p element that is an immediate descendant of a div element
div + p Selects the first p element that is an immediate sibling after a div element

CSS Color Types

Example Color Type
color: red Using red which is one of the 147 'named' format colors available
color: #FF0000 Hexadecimal format for red
color: rgb (255, 160, 122) rgb (red, green, blue) format for red
color: hsl (0, 100%, 50%) hsl (hue, saturation, light) format for red
rgba Adds the "alpha" number to rgb which allows setting of transparency. 0 = transparent, 1 opaque
hsla Adds the "alpha" number to hsl which allows setting of transparency. 0 = transparent, 1 opaque
